# Your Next Meal

A modern meal discovery web app built with React, TypeScript, and Tailwind CSS.

## Author

Created and maintained by **nidhiatwork01-cmyk**.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Run locally

```sh
git clone https://github.com/nidhiatwork01-cmyk/nextmealassign.git
cd nextmealassign
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Create production build
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run test` - Run tests
